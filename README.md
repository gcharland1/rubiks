# Rubiks Cube - V1.0

## How to run
- Setup a venv
- Install requirements.txt
- `python3 main.py`

## How to use the app

### Rotate the image
- Use the arrow keys to rotate the view

### Turn the faces
- Use the `R, L, F, B, U, D` keys to turn clockwisew
- Press shift+key to move counter-clockwise

### Quit the app
- Press Ctrl+Q
